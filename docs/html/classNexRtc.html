Descrição detalhada

Interface com o RTC interno do Nextion.

O Nextion possui um RTC interno (sem bateria de backup). Utilize escreverHora() para sincronizar via NTP ou RTC externo no boot.

Uso típico
NexRtc rtc;
// Sincronizar com NTP (após getLocalTime)
struct tm ti;
if (getLocalTime(&ti)) {
rtc.escrever(ti.tm_year + 1900, ti.tm_mon + 1, ti.tm_mday,
ti.tm_hour, ti.tm_min, ti.tm_sec);
}
// Ler a hora atual
uint32_t h, m, s;
rtc.lerHora(h, m, s);
