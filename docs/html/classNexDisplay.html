Descrição detalhada

Gerenciador principal do display Nextion.

Encapsula inicialização, lista de escuta de toque, processamento do loop e navegação de páginas.

◆ escutar()

bool NexDisplay::escutar ( NexToque &  componente)

Registra um componente para receber eventos de toque.

O componente passado começa a ter seus callbacks disparados automaticamente pelo atualizar().

Parâmetros
componenteReferência para o componente a escutar.
Retorna
true se registrado com sucesso.
false se a lista estiver cheia (NEX_MAX_COMPONENTES).
Exemplo
display.escutar(btnLigar);
display.escutar(btnDesligar);
display.escutar(sliderVolume);
