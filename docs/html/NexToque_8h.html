Descrição detalhada

Componentes com suporte a eventos de toque.

NexToque estende NexObjeto adicionando a capacidade de registrar callbacks para eventos de pressionar e soltar.

Modos de uso

Callback simples (lambda — recomendado):

NexBotao btn(0, 2, "b0");
btn.aoPressionar([]() { Serial.println("Pressionado!"); });
btn.aoSoltar([]() { Serial.println("Solto!"); });

void minhaFuncao(void* ptr) {

Integração com NexDisplay

Para que os callbacks sejam chamados automaticamente no loop(), registre os componentes no NexDisplay:

NexDisplay display;
NexBotao btn(0, 2, "b0");
void setup() {
display.begin(Serial2, 115200, 16, 17);
display.escutar(btn);
btn.aoSoltar([]() { Serial.println("Clicou!"); });
}
void loop() {
display.atualizar(); // processa eventos de toque
}
