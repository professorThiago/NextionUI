Define um atributo numérico do componente.

Envia o comando <nome>.<attr>=<valor> seguido de ref <nome> para atualizar o display. Usa buffer fixo — sem alocação de heap.

Parâmetros
attrNome do atributo Nextion (ex: "bco", "pco", "val").
valorValor numérico a definir.
Retorna
Referência a este objeto (permite encadeamento).
Atributos numéricos comuns
Atributo Significado
val Valor numérico do componente
bco Cor de fundo (RGB565)
pco Cor do texto (RGB565)
font Índice da fonte
xcen Alinhamento horizontal (0-2)
ycen Alinhamento vertical (0-2)
en Habilitado (1) / desabilitado (0)
Exemplo
txt.set("bco", 1024); // muda cor de fundo
txt.set("pco", 65535); // muda cor do texto
num.set("val", 42); // muda valor
timer.set("en", 1); // habilita timer
