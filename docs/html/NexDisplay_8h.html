Descrição detalhada

Ponto de entrada único para o display Nextion.

NexDisplay centraliza toda a inicialização e o processamento do loop. Em vez de chamar funções globais espalhadas, você usa um único objeto com uma API coesa.

Comparação: antes × depois

Antes (ITEADLIB original):

// Funções globais espalhadas
nexInit(115200, 16, 17, &Serial2);
NexTouch *lista[] = { &btn0, &btn1, NULL };
void loop() {
nexLoop(lista);
}

Depois (NextionUI):

NexDisplay display;
NexBotao btn0(0, 2, "b0");
NexBotao btn1(0, 3, "b1");
void setup() {
display.begin(Serial2, 115200, 16, 17);
display.escutar(btn0);
display.escutar(btn1);
}
void loop() {
display.atualizar(); // tudo em um lugar
}
